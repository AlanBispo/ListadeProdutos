# Sobre o projeto

## Listagem de produtos (sem banco de dados)
Projeto de listagem de produtos com as funções de insersão e remoção. Fazendo a comunicação do back-end com o front-end através do axios.

# Tecnologias utilizadas

## Front-end
- HTML / CSS / JavaScript
- ReactJS
## Back-end
- NodeJS
